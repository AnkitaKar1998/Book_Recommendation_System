About Project

  • This book recommendation project is a mini project implemented using the Book Crossing Data available here.
  • On viewing a specific book, details of that book and list of recommended books is shown to the user.
  • Hybrid recommendation algorithms are used in this project which include Collaborative Filtering using Pearson Correlation as well as KNN algorithm and Content Based recommendation algorithm.
  • Since this is a mini project for demonstration, user login data as well as user view history data is not maintained.
  • The recommendation of each book viewed is based only on the ratings of users and other features of that book in the dataset. Current user view actions are not saved and hence not considered for recommendation.
  • The project can be further extended by creating a register/login system and a rating system where the logged in users can give rating to each book. The recommendations can be then be evaluated based user history and new ratings.
  • The source code of the project is available in my GitHub repository here.
